## ğŸ“š What is CO-STAR?

CO-STAR is a prompt engineering framework developed by [GovTech Singapore](https://www.tech.gov.sg/) that helps structure effective AI prompts:

| Element       | Description                          |
| ------------- | ------------------------------------ |
| **C**ontext   | Background information the AI needs  |
| **O**bjective | What you want the AI to accomplish   |
| **S**tyle     | Writing style (formal, casual, etc.) |
| **T**one      | Emotional tone of the response       |
| **A**udience  | Who the output is intended for       |
| **R**esponse  | Desired format of the output         |
